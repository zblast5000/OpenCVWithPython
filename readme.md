# Detector de padroes em imagens MEV 

Programa para detectar padroes em imagens MEV.  
Usar Python 3

## Pacotes utilzados
OpenCV  
Numpy

como instalar via pip
```bash
pip install opencv-python
pip install numpy
```
ou via pip3
```bash
pip3 install opencv-python
pip3 install numpy
```
